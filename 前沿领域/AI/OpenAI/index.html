Ollama 开源大模型 ​

文档

Ollama 是一个开源的大型语言模型（LLM）服务工具，它允许用户在本地机器上运行和部署大型语言模型。

  • 推荐 1：Ollama 官网
  • 推荐 2：Ollama Github 仓库

提示

  • ollama命令行工具的使用很类似docker命令，通过 olloma -h 可查看帮助
  • 推荐阅读：博客文章

列出已有模型 ​

bash
$ ollama list
1

创建模型 ​

ollama create用于从 Modelfile 创建模型

bash
$ ollama create mymodel -f ./Modelfile
1

拉取模型 ​

拉取本地已经存在的模型会自动比较两者差异进行更新

bash
$ ollama pull llama3.2
1

删除模型 ​

bash
$ ollama rm llama3.2
1

复制模型 ​

bash
$ ollama cp llama3.2 my-model
1

显示模型信息 ​

bash
$ ollama show llama3.2
1

列出当前加载的模型 ​

bash
$ ollama ps
1

停止当前正在运行的模型 ​

bash
$ ollama stop llama3.2
1

命令行启动 Ollama ​

bash
$ ollama serve
1
